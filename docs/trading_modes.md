# 장중매매와 종가베팅 모드

브레이브사운드는 한 가지 매매만 강제하지 않는다. 사용자의 실제 매매 흐름에 맞춰 **장중매매 모드**와 **종가베팅 모드**를 분리한다.

## 1. 장중매매 모드

장중매매는 오늘 시장에서 지금 돈이 들어오는 곳을 찾는 모드다.

### 핵심 질문

- 지금 거래대금이 붙고 있는가?
- 주도 섹터 안에서 움직이는가?
- 대장주인가, 후발주인가?
- 시초 갭이 과하지 않은가?
- 눌림 후 다시 살아나는가?
- VWAP 또는 당일 평균단가를 회복했는가?
- 당일 고가권을 유지하고 있는가?

### 좋은 장중매매 후보

- 당일 거래대금 상위 50위 이내
- 대장주 또는 대장 후보
- 시초 급등 후 무너지지 않음
- 첫 눌림에서 지지 확인
- 재상승 때 거래대금 재유입
- 테마 내 동반 상승 유지

### 피해야 할 장중매매

- 장 초반 갭 과열 후 거래대금 식는 종목
- 대장주가 아닌 후발주 추격
- 눌림이 아니라 이탈 중인 종목
- 5일선·20일선 모두 아래인 종목
- 이유 없는 저시총 급등주

## 2. 종가베팅 모드

종가베팅은 오늘 끝까지 강했고, 내일도 재료가 이어질 가능성이 있는 종목을 찾는 모드다.

### 핵심 질문

- 오늘 주도 섹터였는가?
- 당일 또는 전일 거래대금 상위 50위 이내인가?
- 고점 대비 덜 빠졌는가?
- 종가 부근에서 힘이 남아 있는가?
- 내일도 해석될 재료가 있는가?
- 종가 위치가 5일선·20일선 위인가?

### 좋은 종가베팅 후보

- 장중 내내 주도 섹터 안에 있음
- 종가 부근에서 밀리지 않음
- 거래대금이 유지됨
- 대장주가 무너지지 않음
- 익일에도 뉴스 명분이 살아 있음
- 고점 대비 조정이 얕음

### 피해야 할 종가베팅

- 장 막판 억지 반등
- 고점 대비 크게 무너진 뒤 기술적 반등만 나온 종목
- 재료가 당일 소멸된 종목
- 대장주가 무너진 섹터의 후발주
- 다음 날 갭하락 시 손절 기준이 없는 종목

## 3. 실행 방법

전체 비교:

```bash
python src/main.py data/sample_watchlist.csv --mode all --top 10
```

장중매매 후보만 보기:

```bash
python src/main.py data/sample_watchlist.csv --mode intraday --top 10
```

종가베팅 후보만 보기:

```bash
python src/main.py data/sample_watchlist.csv --mode closing --top 10
```

## 4. 입력 데이터 컬럼

기본 필수 컬럼:

```text
ticker,name,sector,market_cap_억,trading_value_rank_today,trading_value_rank_yesterday,high_gap_pct,theme_count,news_score,leader_score,close_above_ma5,close_above_ma20,volume_change_pct
```

추가 선택 컬럼:

```text
intraday_strength_score,pullback_quality_score,vwap_reclaim,near_day_high,closing_strength_score,next_day_catalyst_score,closing_position_score
```

추가 컬럼이 없으면 기본값으로 계산된다. 다만 실제 매매 정확도를 높이려면 장중 강도, 눌림 질, 종가 위치, 익일 재료 점수를 직접 입력하는 편이 좋다.

## 5. 운용 원칙

장중매매는 빠른 판단이 필요하므로 조건이 하나라도 크게 어긋나면 제외한다. 종가베팅은 다음 날까지 보유하는 전략이므로 재료 지속성과 종가 위치를 더 엄격히 본다.

따라서 같은 종목이라도 장중매매에서는 A가 나오고 종가베팅에서는 B가 나올 수 있다. 반대로 장중에는 애매했지만 종가까지 버티면 종가베팅 후보로 올라올 수 있다.
