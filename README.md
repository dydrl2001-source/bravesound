# 브레이브사운드 BraveSound

> 시장의 소리를 듣고, 돈이 몰리는 곳만 선별하기 위한 데이트레이딩·단기스윙 보조 프로젝트입니다.

브레이브사운드는 자동매매 프로그램이 아닙니다. 이 저장소의 목적은 매수 버튼을 대신 누르는 것이 아니라, 매일 반복되는 판단을 규칙화하여 **주도 섹터, 주도주, 거래대금, 뉴스 명분, 차트 위치, 리스크 관리**를 한 화면에서 점검하도록 만드는 것입니다.

## 핵심 문제의식

단타에서 손실이 반복되는 이유는 대체로 기법을 몰라서가 아니라, 장중에 기준이 흔들리기 때문입니다.

- 돈이 몰리는 섹터가 아닌 곳을 매매한다.
- 거래대금과 등락률을 함께 보지 않고 한쪽만 본다.
- 시초가 갭이 과한 종목을 추격한다.
- 눌림이 아니라 무너지는 종목을 산다.
- 손절·분할익절 기준 없이 감정적으로 대응한다.
- 복구매매, 물타기, 집착 매매로 하루의 리스크를 키운다.

브레이브사운드는 이 문제를 줄이기 위해 만든 규칙 기반 매매 보조 시스템입니다.

## 기본 철학

1. **시장 먼저, 섹터 다음, 종목은 마지막에 본다.**
2. **거래대금은 힘이고, 등락률은 현재 시장의 시선이다.**
3. **주도주는 많이 오른 종목이 아니라 돈이 계속 머무는 종목이다.**
4. **시초 추격보다 9:00~9:10 수급 확인, 9:10~9:30 대장 확인, 10시 이후 눌림 확인을 우선한다.**
5. **종가베팅은 확신이 아니라 다음 날 시초 대응을 위한 후보 압축이다.**
6. **모든 매매는 진입 전 손절선과 1차 익절선을 먼저 정한다.**

## 1차 MVP 기능

현재 1차 목표는 아래 조건을 만족하는 종목을 A급, B급, 제외로 분류하는 것입니다.

### 필수 필터

- 당일 또는 전일 거래대금 상위 50위 이내
- 시가총액 1,000억 원 이상 4조 원 이하
- 주도 섹터 소속 여부
- 같은 테마 내 3종목 이상 동반 상승 여부
- 뉴스·재료의 지속 가능성
- 고점 대비 덜 빠진 위치
- 5일선·20일선 위 안착 여부
- 과도한 시초 갭 또는 이유 없는 급등 배제

### 선호 패턴

- 기준봉 발생 후 거래량이 줄어드는 눌림
- 20일선 우상향 중 5일선 재돌파
- 60일 신고가 또는 전고점 돌파 이후 건강한 눌림
- 대장주가 무너지지 않고 후발주만 산발적으로 튀지 않는 구조
- 거래대금 증가와 뉴스 명분이 함께 있는 종목


## 등급 해석

| 등급 | 의미 | 대응 |
|---|---|---|
| A+ | 주도 섹터·거래대금·재료·차트 위치가 강하게 일치 | 관심 최상위. 단, 추격보다 눌림/지지 확인 |
| A | 매매 후보로 충분하나 일부 조건 확인 필요 | 분할 진입 또는 조건부 진입 |
| B | 관찰 가능하지만 확신 부족 | 장중 수급 확인 전까지 보류 |
| WATCH | 아직 매매보다 추적이 우선 | 섹터 확산·거래대금 재유입 확인 |
| EXCLUDE | 핵심 조건 미달 | 매매 제외 |

## 금지 원칙

- 이유 없는 상한가·저시총·우선주 추격 금지
- 시초 갭 과열 종목 무조건 추격 금지
- 손절선 없는 진입 금지
- 물타기와 복구매매 금지
- 대장주가 무너진 테마의 후발주 추격 금지
- 하루에 여러 전략을 동시에 섞는 것 금지

## 현재 단계

이 저장소는 1차로 **규칙 문서화 + 간단한 CSV 스크리너** 단계입니다. 다음 단계에서는 실제 OHLCV 데이터, 거래대금 순위, 섹터 분류, 뉴스 점수, 복기 일지를 연결할 수 있습니다.
